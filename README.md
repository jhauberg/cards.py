## Description

Generates printable sheets of cards.

Currently only A4 pages with Poker-sized cards (2.5x3.5 inches) is supported.

### Browser support

  * Safari
  * Chrome

## Usage

Run from command line:

    $ python cards.py -f example/my-data.csv -t example/my-card-template.html
